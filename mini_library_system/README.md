**📚 Mini Library Management System**
---
A simple **Library Management System** built with **PHP, MySQL, and Tailwind CSS**.  
---

## 🚀 Features
- **Student Management**
  - Add new students
  - Display student list
- **Book Management**
  - Add new books
  - Search/filter books
- **Borrowing System**
  - Select student and book to borrow
  - Delete borrowing records
